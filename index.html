Adverb Clause of Result

Guide Mindmap Examples Quiz
📖

บทเรียนเจาะลึกคำเชื่อมบอกผลลัพธ์สไตล์ปีศาจแดง

Adverb Clause of Result คือ อนุประโยค (ประโยคย่อย) ที่ทำหน้าที่เป็นส่วนขยายกริยาหรือประโยคหลัก เพื่อระบุหรืออธิบาย "ผลลัพธ์ที่เกิดขึ้นตามมา" จากเหตุการณ์ ความรุนแรง หรือปริมาณที่ถูกระบุไว้ในประโยคหลัก

ในแง่ของไวยากรณ์ภาษาอังกฤษ ตัวขับเคลื่อนระบบนี้จะมีจุดสังเกตเด่นชัดคือการทำงานร่วมกับคำเชื่อมสองตัวหลัก ได้แก่ so...that และ such...that ซึ่งแปลไทยว่า "มากเสียจนกระทั่งว่า"

🔴 1. เจาะลึกกลยุทธ์การใช้ SO...THAT

โครงสร้างนี้ถูกออกแบบมาเพื่อเน้นย้ำความรุนแรงของคำคุณศัพท์ (Adjective) หรือคำกริยาวิเศษณ์ (Adverb) ที่อยู่โดดๆ โดยไม่มีคำนามมาต่อท้าย

Subject + Verb + so + [Adj / Adv] + that + Result Clause

วิธีการสังเกตและนำไปใช้: หากในโจทย์สอบปรากฏคำบอกลักษณะ เช่น fast, beautiful, high, expensive ให้เลือกใช้คู่กับ so ทันที

🔴 2. เจาะลึกกลยุทธ์การใช้ SUCH...THAT

ตรงกันข้ามกับ SO ตัวเชื่อมนี้จะถูกส่งลงสนามเมื่อองค์ประกอบด้านหลังมีการรวมตัวกันกลายเป็น "กลุ่มคำนาม (Noun Phrase)" ซึ่งหมายความว่าจะต้องมีคำนามทำหน้าที่เป็นประธานหรือกรรมปิดท้ายวลีเสมอ

Subject + Verb + such + [(a/an) + Adj + Noun] + that + Result Clause

วิธีการสังเกตและนำไปใช้: ให้ตรวจสอบคำข้างหน้าคำเชื่อม that หากพบคำนามเดี่ยวหรือคำที่มีรูปคำคุณศัพท์นำหน้าคำนาม เช่น a beautiful day, great players, hot weather จุดนี้คือพื้นที่สัมปทานของ such เท่านั้น

🔴 3. กฎเหล็กจุดโทษของกรรมการ (Must-Know)

ข้อควรระวังระดับทีมชาติ: คำบอกปริมาณ (Quantity Words)

นักเรียนมักโดนหลอกในข้อสอบบ่อยครั้งเมื่อเจอคำนามที่มีตัวระบุปริมาณ ได้แก่ many, few, much, little คุมอยู่ด้านหน้า แม้คำเหล่านั้นจะเป็นกลุ่มคำนาม แต่ทางไวยากรณ์บังคับให้ใช้คู่กับ SO เท่านั้น ห้ามใช้ such เด็ดขาด!

so many + นามนับได้พหูพจน์ / so much + นามนับไม่ได้

คลิกเพื่อกางหัวข้อวิเคราะห์แยกประเภทคำเชื่อม

⚽ การวางตําแหน่งประโยค (Positioning)▼
Adverb Clause of Result จะต้องวางต่อท้ายประโยคหลัก (Main Clause) เสมอ เพื่อทำหน้าที่เป็นส่วนรับช่วงต่อเหตุการณ์เพื่อสรุปผลลัพธ์ และในภาษาพูดที่เป็นกันเอง เราสามารถ "ตัดคำว่า that ออกได้" โดยที่ความหมายไม่เปลี่ยนแปลงครับ
⚽ การแปลงประโยคย้อนกลับ (Inversion)▼
ในข้อสอบระดับสูง หากนำคำว่า So + Adj ขึ้นต้นประโยค จะต้องทำการสลักษณ์โครงสร้างประโยคส่วนแรกให้กลายเป็นรูปประโยคคำถาม (Inversion) เช่น So high did he jump that he touched the bar. (เขากระโดดสูงมากเสียจนแตะคาน)
📊

คลังประโยคตัวอย่างแมตช์เดย์จำลอง

The midfielder passed the ball so accurately that it split the defense.
กองกลางส่งบอลแม่นยำมากจนกระทั่งมันทะลุผ่านแนวกองหลังทั้งหมด
so + adverb (accurately)
It was such a heavy rain that the match pitch became flooded.
ฝนตกหนักมากจนกระทั่งพื้นสนามแข่งขันเกิดน้ำท่วมขัง
such + noun phrase (heavy rain)
The striker had so many scoring opportunities that he won the golden boot.
กองหน้ามีโอกาสทำประตูมากมายจนทำให้เขาคว้าตราสโมสรทองคำมาครอง
กฎพิเศษ: so + many + Noun
The pitch was not so slippery that the players couldn't run.
พื้นสนามไม่ได้ลื่นมากจนขนาดที่ทำให้นักเตะไม่สามารถวิ่งได้
not + so + adjective
It was not such a serious injury that required surgery.
มันไม่ได้เป็นอาการบาดเจ็บที่รุนแรงมากจนถึงขั้นต้องเข้ารับการผ่าตัด
not + such + noun phrase
Was the tactical plan so complex that the squad got confused?
แผนการเล่นมันซับซ้อนมากจนทำให้คนในทีมเกิดความสับสนใช่ไหม?
Question with SO
Did they offer such a low price that the club rejected the transfer?
พวกเขายื่นข้อเสนอราคาต่ำมากจนสโมสรปฏิเสธการย้ายทีมเลยหรือเปล่า?
Question with SUCH
