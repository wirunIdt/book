Adverb Clause of Result

คำอธิบาย ผังความจำ ตัวอย่าง ทำข้อสอบ
📖

บทเรียนเจาะลึกคำเชื่อมบอกผลลัพธ์สไตล์ปีศาจแดง

Adverb Clause of Result คือ อนุประโยค (ประโยคย่อย) ที่ทำหน้าที่ขยายประโยคหลัก เพื่ออธิบายถึง "ผลลัพธ์ที่เกิดขึ้นตามมาจากเหตุการณ์ก่อนหน้า" โดยมีหลักคิดง่ายๆ คือ เหตุการณ์แรกมีความรุนแรง มีปริมาณ หรือมีความเด่นชัด "มากเสียจนกระทั่งส่งผลให้เกิดเหตุการณ์ที่สองตามมา" ไวยากรณ์บทนี้ควบคุมด้วย 2 ตัวเชื่อมหลักคือ so...that และ such...that

🔴 1. เจาะลึกการใช้ SO...THAT (มากจนกระทั่ง)

โครงสร้างนี้ใช้ขยายคำประเภท คำคุณศัพท์ (Adjective) หรือ คำกริยาวิเศษณ์ (Adverb) ที่วางอยู่โดดๆ ลำพัง โดยไม่มีคำนามตามท้าย

ประธาน + กริยา + so + [คำคุณศัพท์ / คำวิเศษณ์] + that + ประโยคบอกผลลัพธ์

วิธีสังเกตในสนามสอบ:

  • ขยาย Adjective: เช่น The ticket was so expensive that I couldn't buy it. (expensive เป็นคำคุณศัพท์บอกราคาแพง วางอยู่หลัง Verb to be)
  • ขยาย Adverb: เช่น Rashford ran so quickly that defenders fell down. (quickly เป็นคำวิเศษณ์ขยายการวิ่ง ran)
🔴 2. เจาะลึกการใช้ SUCH...THAT (มากจนกระทั่ง)

โครงสร้างนี้ใช้เมื่อกลุ่มคำข้างหลังช่องว่างมีการรวมตัวกันเป็น "กลุ่มคำนาม (Noun Phrase)" เสมอ ไม่ว่าจะเป็นนามเอกพจน์ พหูพจน์ หรือนามนับไม่ได้

ประธาน + กริยา + such + [(a/an) + คุณศัพท์ + คำนาม] + that + ประโยคบอกผลลัพธ์

การจำแนกรูปคำนามอย่างละเอียด:

  • นามนับได้เอกพจน์: ต้องมี a / an นำหน้าเสมอ เช่น such a beautiful goal that... หรือ such an intense match that...
  • นามนับได้พหูพจน์: ห้ามใส่ a/an เด็ดขาด และคำนามต้องเติม s/es เช่น such great players that...
  • นามนับไม่ได้: ห้ามใส่ a/an และห้ามเติม s เช่น such heavy rain that... (ฝนตกหนักมากจน...)
🔴 3. กฎเหล็กจุดโทษระวังโดนหลอก (Must-Know Exceptions)

จุดห้ามพลาด: หากมีคำบอกปริมาณอยู่ด้านหน้าคำนาม

หากข้างหน้าคำนามมีคำบอกปริมาณต่อไปนี้ปรากฏอยู่ ได้แก่ many, few, much, little แม้ข้างหลังจะเป็นคำนาม แต่กฎไวยากรณ์บังคับให้ใช้คู่กับ SO เท่านั้น ห้ามใช้ such เด็ดขาด!

so many + นามพหูพจน์ | so few + นามพหูพจน์ (น้อยจน...) so much + นามนับไม่ได้ | so little + นามนับไม่ได้ (น้อยจน...)

ตัวอย่าง: The club spent so much money that they broke the rules. (ใช้ so คู่กับ much money ห้ามใช้ such)

🔴 4. โครงสร้างแบบสลับประโยคขั้นสูง (Inversion Style)

ในข้อสอบระดับสูง หากต้องการเน้นย้ำผลลัพธ์ให้มีความเป็นทางการและทรงพลัง สามารถนำเอาสำนวน So + คำคุณศัพท์ ขึ้นต้นประโยคได้ แต่ต้องสลับเอาตัวกริยามาวางไว้ข้างหน้าประธานเหมือนรูปประโยคคำถาม

So + [คำคุณศัพท์] + [กริยาช่วย] + ประธาน + that + ...

ตัวอย่างแปลงรูป: So exciting was the match that no one left. (การแข่งขันน่าตื่นเต้นมากเสียจนไม่มีใครยอมเดินออกจากสนาม)

Interactive Mindmap แผนผังช่วยจำทางยุทธวิธี

ผู้สูงอายุสามารถคลิกที่หัวข้อด้านล่างเพื่อกางอ่านเนื้อหาเพิ่มเติมได้ทันทีครับ

⚽ การตัดคำว่า That ทิ้งในภาษาพูด▼
ในทางปฏิบัติที่ไม่เป็นทางการหรือการพูดคุยทั่วไป เราสามารถละคำว่า that ออกไปได้โดยความหมายไม่เปลี่ยนแปลง เช่น The stadium was so loud (that) I couldn't hear. แต่ถ้าทำในข้อสอบโรงเรียนแนะนำให้เขียนคำว่า that ไว้เสมอเพื่อความปลอดภัยครับ
⚽ ความแตกต่างระหว่าง So...that กับ Too...to▼
โครงสร้างต่างกันโดยสิ้นเชิง So...that จะตามหลังด้วยประโยคเต็มรูปที่มีประธานและกริยาครบถ้วน แต่ Too + คุณศัพท์ + to + กริยาช่อง 1 จะแปลว่า "มากเกินกว่าที่จะทำได้" ซึ่งมีความหมายติดลบและทำไม่ได้อยู่ในตัวอยู่แล้ว
⚽ สูตรลัดจำง่ายเอาไว้ใช้ทำข้อสอบความเร็วสูง▼
1. มองไปหลังช่องว่าง -> เจอคำคุณศัพท์โดดๆ ไม่มีนามค้ำท้าย เลือกตอบ SO
2. มองไปหลังช่องว่าง -> เจอคำนามต่อท้ายมาด้วย เลือกตอบ SUCH
3. มองไปหลังช่องว่าง -> เจอคำแก๊งปริมาณ many / few / much / little เลือกตอบ SO เสมอ!
📊

คลังประโยคตัวอย่างจำลองเหตุการณ์ในแมตช์การแข่งขัน

The midfielder passed the ball so accurately that it split the defense.
กองกลางส่งลูกบอลอย่างแม่นยำมากจนกระทั่งมันสามารถเจาะทะลุแนวกองหลังได้
ใช้ so + ขยายกริยาวิเศษณ์ (accurately)
It was such a heavy rain that the match pitch became flooded.
มันเป็นฝนที่ตกหนักมากจนกระทั่งพื้นสนามแข่งขันเกิดน้ำท่วมขัง
ใช้ such + ตามด้วยกลุ่มนามเอกพจน์ (heavy rain)
The striker had so many scoring opportunities that he easily won the golden boot.
กองหน้ามีโอกาสทำประตูมากมายจนทำให้เขาคว้าสโมสรทองคำมาครองได้อย่างง่ายดาย
กฎพิเศษ: so + วางหน้าคำว่า many
They are such loyal fans that they never leave the stadium before the final whistle.
พวกเขาเป็นแฟนบอลที่ภักดีมากเสียจนไม่เคยเดินออกจากสนามก่อนนกหวีดหมดเวลา
ใช้ such + กลุ่มนามพหูพจน์ (loyal fans)
Bruno Fernandes ran so tirelessly that he covered over 12 kilometers in one match.
บรูโน่ แฟร์นันเดส วิ่งอย่างไม่รู้จักเหน็ดเหนื่อยมากจนกระทั่งเขาวิ่งระยะทางเกิน 12 กิโลเมตรในนัดเดียว
ใช้ so + ขยายกริยาวิเศษณ์ (tirelessly)
The young prospect displayed such incredible talent that Europe's biggest clubs made offers.
นักเตะดาวรุ่งแสดงพรสวรรค์ที่น่าทึ่งมากจนสโมสรยักษ์ใหญ่ในยุโรปพากันยื่นข้อเสนอ
ใช้ such + กลุ่มนามนับไม่ได้ (incredible talent)
The pitch was not so slippery that the players couldn't control the ball.
พื้นสนามไม่ได้ลื่นมากจนขนาดที่ทำให้นักเตะไม่สามารถควบคุมลูกบอลได้
รูปปฏิเสธ: not + so + คุณศัพท์
It was not such a serious injury that required major surgery.
มันไม่ได้เป็นอาการบาดเจ็บที่รุนแรงมากจนถึงขั้นต้องเข้ารับการผ่าตัดใหญ่
รูปปฏิเสธ: not + such + นามวลี
The team had so little possession that they couldn't create a single shot on target.
ทีมมีเปอร์เซ็นต์การครองบอลน้อยมากจนไม่สามารถสร้างโอกาสยิงเข้ากรอบได้เลย
กฎพิเศษปริมาณน้อย: so + little + นามนับไม่ได้
The manager's press conference was not so controversial that it caused a team drama.
การแถลงข่าวของผู้จัดการทีมไม่ได้เป็นประเด็นขัดแย้งมากจนทำให้เกิดเรื่องดราม่าในทีม
รูปปฏิเสธใช้คู่กับ SO
The referee did not make such obvious errors that the committee needed to suspend him.
ผู้ตัดสินไม่ได้ทำความผิดพลาดที่ชัดเจนมากจนถึงขั้นที่คณะกรรมการต้องสั่งพักงาน
รูปปฏิเสธใช้คู่กับ SUCH + นามพหูพจน์
So powerful was the shot that the ball ripped through the stadium net.
ลูกยิงนั้นทรงพลังมากเสียจนลูกบอลพุ่งทะลุตาข่ายของสนามแข่งขันออกไปเลย
โครงสร้างกลับประโยคขั้นสูง (So + คุณศัพท์ + กริยา + ประธาน)
Was the tactical plan so complex that the squad got confused?
แผนการเล่นมันซับซ้อนมากจนทำให้คนในทีมเกิดความสับสนใช่ไหม?
รูปประโยคคำถามที่ใช้ SO
Did they offer such a low price that the club rejected the transfer immediately?
พวกเขายื่นข้อเสนอราคาต่ำมากจนสโมสรปฏิเสธการย้ายทีมในทันทีเลยหรือเปล่า?
รูปประโยคคำถามที่ใช้ SUCH
So fast did the winger sprint that the assistant referee missed the offside line.
ปีกตัวรุกสปริ้นท์ตัววิ่งเร็วมากจนผู้ช่วยผู้ตัดสินมองเส้นล้ำหน้าไม่ทัน
โครงสร้างกลับประโยคขั้นสูงสไตล์คำวิเศษณ์
